## 3. Аннотация (реферат)

Платформа Hyppo (дистрибутив `gedanken`) предназначена для управления
семействами конкурирующих научных гипотез в форме воспроизводимых виртуальных
экспериментов. Программа автоматически строит решётку гипотез из
уравнительных структур гипотез методом причинного упорядочивания
(Алгоритм 1) и поддерживает её инкрементальное обновление при добавлении
новых гипотез без полной перестройки (Алгоритм 2, с доказанной
эквивалентностью полной перестройке). Двухэтапная процедура проверки
корректной определённости сочетает структурный анализ решётки с OWL-выводом
через рассуждатель (Алгоритм 3). Для выполнения эксперимента строится
минимальный план пересчёта фрагментов с использованием кэша ранее
вычисленных результатов, для которого доказаны корректность и
⊆-минимальность (Алгоритм 4, Теорема 1). Исполнение плана сопровождается
присвоением гипотезам эпистемических статусов (PROPOSED/SUPPORTED/REFUTED/
SUPERSEDED) и статистическим сравнением конкурирующих гипотез (знаковый
критерий, AIC/BIC). Платформа включает веб-интерфейс (FastAPI + React/
TypeScript) для визуализации решётки гипотез и планов и MCP-сервер для
взаимодействия с ИИ-агентами. Область применения — научные исследования с
множественными конкурирующими объяснительными моделями, в частности
нефтегазовая отрасль (прогнозирование добычи жидкости и нефти при
заводнении пласта, задачи материального баланса и обводнённости).

## 4. Тип реализующей ЭВМ

IBM PC-совместимый персональный компьютер / сервер.

## 5. Вид и версия операционной системы

Windows, Linux, macOS (кроссплатформенная реализация на Python; требований к
конкретной версии ОС нет за пределами поддержки интерпретатора Python 3.11+).

## 6. Язык программирования

Python 3.11+ (основная реализация: ядро платформы, алгоритмы 1–4, OWL-модель,
веб-API); TypeScript/React (клиентская часть веб-интерфейса, компилируется в
статические ассеты, поставляемые вместе с пакетом).

## 7. Объём программы

| Компонент | Файлов | Строк | Объём |
|---|---|---|---|
| Ядро платформы `hyppo/` (Python, *.py) | 75 | 7 068 | ≈ 250 КБ |
| Веб-интерфейс `hyppo/gui/static/` (собранные TS/React-ассеты + HTML) | 3 | — | ≈ 232 КБ |
| **Итого** | **78** | **≈ 7 068 (Python) + сборка фронтенда** | **≈ 482 КБ** |

Объём исходного кода ядра платформы (пакет `hyppo`) — 7068 строк Python в 75
файлах, суммарно около 250 КБ текста. Отдельно поставляется собранный
веб-интерфейс (`hyppo/gui/static/`, 3 файла: `index.html` и JS/CSS-бандлы
React-приложения) объёмом около 232 КБ. Депонируемый листинг (см.
`docs/rid/listing.md`) включает полные исходные тексты ключевых модулей,
реализующих Алгоритмы 1–4 и Теорему 1, а также сводную таблицу объёма
остальных модулей пакета.
