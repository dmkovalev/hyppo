# hyppo.adapters

Адаптеры, соединяющие платформу виртуальных экспериментов Hyppo с внешними системами. Основной пример — адаптер к нефтяному ВЭ-кейсу Norne/Brugge (`norne_adapter`) для виртуального эксперимента по заводнению нефтяного пласта: функция `build_oil_virtual_experiment` строит OWL DL-представления шести суб-гипотез гибридной модели CRM (capacitance-resistance model) и отображает их на дискретное пространство гиперпараметров настройки модели (см. раздел 4.10 диссертации), а `run_oil_experiment_demo` запускает демонстрационный прогон такого эксперимента. Важно, что модуль не импортирует какой-либо внешний пакет оптимизации или симуляции пласта — он строит исключительно онтологические артефакты, зеркалирующие архитектуру модели HybridCRM, что позволяет проводить формальный вывод (reasoning) над структурой эксперимента независимо от исполняемого кода. Пакет служит примером того, как доменные структуры прикладной задачи (нефтепромысловая инженерия) выражаются в терминах решётки гипотез и онтологии `hyppo.core`/`hyppo.ontology`.
