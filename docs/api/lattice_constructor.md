# hyppo.lattice_constructor

Пакет построения решётки гипотез из структур виртуального эксперимента. Основной класс `HypothesisLattice` принимает набор гипотез и описание рабочего процесса (workflow) и строит решётку методом `build_lattice`, реализующим Алгоритм 1 диссертации: ребро `(h_i, h_j)` проводится, когда выходная переменная гипотезы `h_i` входит в число входных переменных уравнений гипотезы `h_j` (то есть `h_j` производна от `h_i`). Класс встроен в OWL-онтологию виртуального эксперимента (`virtual_experiment_onto` из `hyppo.core`), что позволяет применять к решётке правила OWL DL из `hyppo.ontology` для проверки согласованности и качества гипотез. Построенная решётка гипотез служит входом для планировщика (`hyppo.planner`), который на её основе определяет, какие гипотезы требуют пересчёта, а какие можно взять из кеша репозитория метаинформации.

Подробная справка по сигнатурам генерируется из docstrings (на английском):

::: hyppo.lattice_constructor
