# hyppo.versioning

Пакет версионирования гипотез: контент-адресуемые снимки (snapshots) состояния гипотез и провенанс их запусков. Подмодуль `_db` определяет модели SQLAlchemy и движок базы данных, в том числе таблицы `HypothesisVersion` (версии гипотез) и `HypothesisRunLink` (связь версии гипотезы с конкретным запуском исполнителя); подмодуль `version_store` предоставляет асинхронные функции-обёртки для сохранения версий, регистрации связей запусков и запросов по истории версионирования — каждая функция открывает асинхронную сессию, выполняет запрос и фиксирует изменения при необходимости. Пакет намеренно сделан независимым от `hyppo.mcp` и `hyppo.actions`: зависимость идёт в обратную сторону — `hyppo.actions` использует `hyppo.versioning`, а не наоборот, что позволяет избежать цикла импортов. Версионирование гипотез обеспечивает воспроизводимость: для каждой сохранённой версии можно восстановить, при каком запуске эксперимента и с какими входными данными она была получена.

Подробная справка по сигнатурам генерируется из docstrings (на английском):

::: hyppo.versioning
