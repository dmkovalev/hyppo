# hyppo.gui

Веб-интерфейс платформы Hyppo на базе FastAPI. Функция `create_app` собирает приложение `FastAPI`, инициализирует хранилище проектов `ProjectStore` (с указанным путём к базе данных `db_path`), заполняет его демонстрационными данными через `seed_demo` и подключает маршрутизаторы API: `projects` (управление проектами виртуальных экспериментов) и `hypotheses` (просмотр и работа с гипотезами и их решёткой). Эндпоинт `/api/health` предоставляет простую проверку работоспособности сервиса. GUI играет роль тонкого визуального слоя поверх основной логики hyppo (`hyppo.manager`, `hyppo.lattice_constructor`, `hyppo.runner`) — он не содержит собственной научной логики, а лишь предоставляет пользователю обзор проектов, решёток гипотез и их эпистемических статусов через HTTP API.
