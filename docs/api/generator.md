# hyppo.generator

Генератор гипотез из данных (`HypothesisGenerator`), описанный в разделе 3.1.3 диссертации. Функция `generate_hypothesis` реализует упрощённую версию пятистадийного конвейера полного метода: (1) GLM — извлечение значимых комбинаций переменных линейной регрессией; (2) DCM — извлечение производных (в минимальной реализации пропускается); (3) GP — символьная регрессия (опционально, требует `deap`); (4) NeuralODE (в минимальной реализации пропускается); (5) ранжирование кандидатов по коэффициенту детерминации R². На вход подаются матрица признаков `X`, вектор отклика `y` и минимальный порог R² (`r2_min`); на выходе — список пар (текстовое описание гипотезы, значение R²), отсортированный по убыванию R² и отфильтрованный по порогу. Сгенерированные таким образом гипотезы становятся кандидатами для построения решётки гипотез (`hyppo.lattice_constructor`) и последующей оценки в виртуальном эксперименте.

Подробная справка по сигнатурам генерируется из docstrings (на английском):

::: hyppo.generator
