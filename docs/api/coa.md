# hyppo.coa

Пакет алгоритмов причинного упорядочивания (Causal Ordering Algorithms), реализующий Алгоритмы 1, 2 и 4 диссертации. Ключевой класс `HypothesisGraph` представляет решётку гипотез как DAG над отношением `derived_by`: узлы — это гипотезы (структуры вида `list[frozenset[str]]` над именами переменных), рёбра `i -> j` означают, что гипотеза `j` выводится из `i`. Класс предоставляет методы для построения решётки (Алгоритм 1), причинного упорядочивания вершин и определения множеств гипотез, требующих пересчёта либо допускающих переиспользование кеша (Алгоритм 2/4). Всё причинное упорядочивание делегируется модулю `hyppo.coa.causal`, реализующему декомпозицию Дюльмажа-Мендельсона (паросочетание Куна + компоненты сильной связности по алгоритму Тарьяна) — таким образом измеряемая сложность отражает саму библиотеку, а не её переупрощённую реализацию в тестовом скрипте. Слой `hyppo.coa.graph` не тянет зависимостей sympy/owlready и может использоваться отдельно; более богатая sympy-версия `hyppo.coa._base.Structure` подгружается лениво.

Подробная справка по сигнатурам генерируется из docstrings (на английском):

::: hyppo.coa
