# hyppo.ontology

Пакет правил OWL 2 DL для платформы виртуальных экспериментов Hyppo. Реализует 16 правил OWL DL (без SWRL и арифметики), расширяющих базовую онтологию из `hyppo.core._base` (Алгоритм 3 диссертации). Правила сгруппированы по назначению: базовая классификация и структурные аксиомы (правила 1-6, классы вроде `PhysicsHypothesis`, `DataDrivenHypothesis`, `HybridHypothesis`, `ValidHypothesis`, `InvalidHypothesis`); провенанс и версионирование (правила 7-8); валидация рабочего процесса (правила 9-10); фильтры качества для отсечения решётки гипотез (`StaleHypothesis`, `UncomputedHypothesis`, правила 11-12); обнаружение совместного использования между экспериментами (правило 13); проверка совместимости моделей и форматов данных (правила 14-15); управление состояниями жизненного цикла (правило 16). Модуль `consistency` предоставляет функцию `check_consistency`, возвращающую `ConsistencyResult` со статусом согласованности онтологии, а дополнительный модуль `oil_constraints` содержит физические валидаторы на Python для проверок, невыразимых в чистом OWL DL (например, доменных ограничений нефтепромысловой предметной области).

Подробная справка по сигнатурам генерируется из docstrings (на английском):

::: hyppo.ontology
