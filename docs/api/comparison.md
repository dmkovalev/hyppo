# hyppo.comparison

Пакет статистических методов сравнения и ранжирования гипотез. Предоставляет набор тестов и метрик для сопоставления качества конкурирующих гипотез/моделей: непараметрические критерии `sign_test` (знаковый тест) и `wilcoxon_test`/`pairwise_wilcoxon_by` (тест Уилкоксона, в том числе для попарного сравнения по группам), поправку на множественные сравнения `benjamini_yekutieli`, информационные критерии `compute_aic` и `compute_bic`, функцию гауссовского логарифмического правдоподобия `gaussian_log_likelihood`, байесовский апостериорный расчёт `bayesian_posterior` и итоговую комбинированную функцию ранжирования `combined_ranking`, агрегирующую перечисленные метрики в единый порядок предпочтения гипотез. Эти инструменты применяются после исполнения виртуального эксперимента (`hyppo.runner`) для статистически обоснованного выбора наилучшей гипотезы среди тех, что получили эпистемический статус `SUPPORTED`, и лежат в основе правила `SUPERSEDED` (ΔAIC > θ_aic) из `hyppo.core`.
